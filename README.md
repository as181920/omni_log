# OmniLog
通用日志记录(Rails engine)

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'omni_log', git: "git@github.com:as181920/omni_log.git", branch: "master"
```

And then execute:
```bash
$ bundle
```
